# VehiclePredict-SL

AI-Powered Vehicle Registration Forecasting for Sri Lanka

## Overview

VehiclePredict-SL is an advanced machine learning system designed to predict vehicle registrations and forecast automotive market trends in Sri Lanka. Using state-of-the-art algorithms and explainable AI, this project provides accurate predictions and actionable insights for the automotive industry.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/VehiclePredict-SL.git
cd VehiclePredict-SL
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python flask_app.py
```

4. Open your browser and navigate to `http://localhost:5000`

## Usage

### Making Predictions
- Navigate to the "Predict" section in the web interface
- Input the relevant vehicle registration parameters
- Get instant predictions with confidence intervals

### Exploring Data
- Access the "Explore" section to visualize historical data
- Analyze market trends and patterns
- Download reports for further analysis

### Understanding Model Decisions
- Visit the "Explainability" section
- Analyze feature importance and model explanations
- Gain insights into what drives predictions
