About RetailPredict

Understanding Sri Lankan retail markets through AI

Project Overview

RetailPredict is an AI-powered application designed to forecast retail prices in Sri Lanka using advanced machine learning techniques. The system analyzes historical market data to provide accurate price predictions and insights into market trends.

Technology Stack

Machine Learning

  • CatBoost: Gradient boosting framework optimized for categorical features
  • SHAP: Explainable AI for understanding model predictions
  • scikit-learn: Data preprocessing and evaluation metrics

Backend

  • Flask: Lightweight Python web framework
  • Python: Primary programming language
  • Pandas: Data manipulation and analysis

Frontend

  • HTML5: Semantic markup
  • CSS3: Modern styling and animations
  • JavaScript: Interactive features
  • Plotly: Interactive data visualization

Model Performance

Algorithm: CatBoost Regressor (Gradient Boosting)
R² Score: 0.9573 (95.73%)
Mean Absolute Error: ₨119.31
Root Mean Square Error: ₨185.24
Training Samples: 5000+ records
Features: 15+ categorical and numeric features

Key Features

  • Real-time Price Predictions: Get instant price forecasts based on market factors
  • SHAP Explainability: Understand why predictions are made with transparent feature importance analysis
  • Data Visualization: Interactive charts showing price distributions, trends, and patterns
  • Market Insights: Explore historical data and identify market trends
  • High Accuracy: 95.73% R² score ensures reliable forecasts

Data Sources

The model has been trained on comprehensive retail price data from Sri Lankan markets, including:

  • Historical price records from multiple markets across regions
  • Product categories including vegetables, fruits, and other retail items
  • Market supply and demand indicators
  • Seasonal and temporal patterns
  • Regional price variations
