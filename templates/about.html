About Vehicle Registration Predictor

Understanding Sri Lankan automotive markets through AI

Project Overview

Vehicle Registration Predictor is an AI-powered application designed to forecast vehicle registrations in Sri Lanka using advanced machine learning techniques. The system analyzes historical automotive market data to provide accurate registration predictions and insights into market trends.

Technology Stack

Machine Learning

  • CatBoost: Gradient boosting framework optimized for categorical features
  • SHAP: Explainable AI for understanding model predictions
  • scikit-learn: Data preprocessing and evaluation metrics

Backend

  • Flask: Lightweight Python web framework
  • Python: Primary programming language
  • Pandas: Data manipulation and analysis

Frontend

  • HTML5: Semantic markup
  • CSS3: Modern styling and animations
  • JavaScript: Interactive features
  • Plotly: Interactive data visualization

Model Performance

Algorithm: CatBoost Regressor with Bayesian Optimization
R² Score: 0.9842 (98.42%)
Mean Absolute Error: ±160 vehicles/month
Optimization Method: Bayesian Hyperparameter Optimization (100 trials)
Training Samples: 648 monthly records (2018-2025)
Features: 10 engineered market and categorical features

Key Features

  • Vehicle Registration Forecasting: Get instant registration predictions based on market conditions
  • SHAP Explainability: Understand why predictions are made with transparent feature importance analysis
  • Data Visualization: Interactive charts showing registration distributions, trends, and patterns
  • Market Insights: Explore historical data and identify automotive market trends
  • High Accuracy: 98.42% R² score ensures reliable forecasts

Data Sources

The model has been trained on comprehensive vehicle registration data from Sri Lanka, including:

  • Historical registration records covering 2018-2025
  • Vehicle categories including cars, motorcycles, auto-rickshaws, and commercial vehicles
  • Market supply and demand indicators
  • Seasonal and temporal patterns
  • Economic growth and policy impacts
